What you can do

Six integrated modules for country comparison and clustering analysis.

🌲

Tree Edit Distance

Zhang-Shasha algorithm computes the minimum cost sequence of edits to transform one country's infobox tree into another.

🔤

Semantic Similarity

Jaccard token overlap across all leaf text values — language, religion, government type, capital city, and more.

⚖️

Combined Scoring

Tune α to blend structural and semantic scores. See exactly how the weighting changes relative rankings.

◎

4 Clustering Algorithms

Agglomerative, DBSCAN, Spectral, and K-Medoids clustering on the pairwise similarity matrix. Visualised as dendrogram, scatter, force graph, and map.

🗺️

Interactive Maps

Select countries by clicking directly on a dark Leaflet world map. Similarity results overlay a geographic heat gradient.

📄

Document Inspector

Explore each country's infobox in Rendered, XML, JSON, and Tree view before running any computation.
